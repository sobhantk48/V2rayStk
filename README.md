# V2ray Stk

Professional dual-panel (Admin + User) multi-protocol VPN client for Android powered by **Sing-box** core.

## Features
- Exactly 16 protocols (VLESS, VMess, Trojan, Shadowsocks, Hysteria2, TUIC, WireGuard, ShadowTLS, AnyTLS, NaïveProxy, Tor, SSH, SOCKS, HTTP, Hysteria, VLESS+XTLS Reality)
- Full Admin Panel (password: `admin` - changeable)
- User Panel with big Connect button (Blue → Green → Red)
- Subscription, Group, Bulk Import/Export, QR Scan, Smart Paste
- Real stats: Ping, GeoIP, Speed, Duration, Traffic history, Speed Test, Quality Test
- Kill Switch, Split Tunneling, Biometric Lock, Multi-Hop, Firewall, DoH/DoT, Lite Mode, Mux, Sniffing, Fragment and all listed advanced features
- Fully bilingual (فارسی + English)
- Built only via GitHub Actions

## Build
Just push to main. GitHub Actions will produce the APKs automatically.

`admin`
