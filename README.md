# V2ray Stk

Flutter client for V2Ray/Xray with Riverpod, an Android Kotlin VPN service,
libbox JNI integration, optional Tor routing, bilingual Persian/English UI,
and role-based Admin/User panels.

Install Flutter and the Android SDK, run `flutter pub get`, then `flutter run`.
Release APKs are produced by `.github/workflows/build-apk.yml`.
